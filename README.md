- `type_alias.cpp` - 类型别名示例
- `namespace_import.cpp` - 命名空间引入示例
- `using_declaration.cpp` - using声明示例
- `using_directive.cpp` - using指令示例
- `template_alias.cpp` - 模板别名示例

## 编译运行
所有示例都需要C++11或更高版本：
```bash
g++ -std=c++11 example.cpp -o example
```
